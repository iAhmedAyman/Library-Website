what you can do in this website?

User Features

Search for Books

Find books by title, author, or category to easily discover what you're looking for.

Book Search Results

View a list of books with key information such as title, author, category, and availability status (borrowed, available for borrowing, or not available).

View Book Details

Select any book to see its full description and additional information.

Borrow Books

Admin Features

Add New Books

Admins can add new books to the system, including all relevant details like title, author, category, and availability.

View List of All Books

Admins can view a list of all books in the system, including those available and borrowed.

Edit Book Details

Admins can select a book and modify its details, such as title, author, availability, and other relevant information.

Delete a Book

Admins can remove books from the system when necessary.
